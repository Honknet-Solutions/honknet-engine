# Honknet Studio

Studio запускается командой `npm run dev:studio`.

Разделы:

- Prototype Builder;
- Component Schema Builder;
- UI/HUI Builder;
- Behavior Graph Builder;
- Map Builder;
- Localization editor.

Редактор работает с обычными YAML и FTL файлами. В браузерах с File System Access API кнопка Save пишет прямо в выбранный файл. В остальных браузерах используется Download.
