# Roadmap

This roadmap describes the route toward the full product, not a reduced MVP.

## Phase 1 — Foundation

- Rust workspace
- TypeScript browser client
- shared protocol crate/package
- WebSocket transport
- basic server tick loop
- entity spawn/despawn
- map and transform primitives

## Phase 2 — Multiplayer World

- movement replication
- interpolation
- interest management
- map chunks
- interaction system
- chat system

## Phase 3 — Framework Layer

- inventory
- hands
- containers
- permissions
- admin commands
- prototype database
- asset pipeline

## Phase 4 — Full Product Systems

- editor
- persistence
- module loading
- scripting/content API
- WebTransport support
- observability
- production deployment tooling
