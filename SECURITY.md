# Security

Не публикуйте уязвимости в открытых issues. Передавайте отчёт владельцам Honknet Solutions приватно.

Сервер является единственным источником истины. Клиентские координаты, инвентарь, UI actions и script-host commands должны проходить серверную валидацию. Production deployment должен ограничивать доступ script-host к файловой системе и сети средствами контейнера или ОС.
