# Honknet Engine v1 RC1

В сборку входят:

- авторитетный Rust-сервер и fixed-tick simulation;
- динамический ECS с компонентами и системами;
- WebSocket handshake, input sequencing, snapshots, reconnect и PVS;
- client prediction/reconciliation и interpolation;
- карта, тайловые коллизии, двери, предметы, контейнерный инвентарь и чат;
- asset manifest и проверка RSI;
- PNG/WebP/RSI sprite pipeline и многослойные спрайты;
- YAML prototype/content loader с наследованием;
- JSON persistence и atomic saves;
- admin command registry;
- sandbox-oriented TypeScript script-host protocol;
- HUI renderer, UI session protocol и behavior graph formats;
- Honknet Studio: prototype, component schema, UI, behavior, map и localization builders;
- пустой SDK и example game module;
- CI и документация.
