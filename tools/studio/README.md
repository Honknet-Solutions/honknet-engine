# @honknet/studio 3.0.0

Визуальная среда разработки контента Honknet Engine.

```bash
npm install
npm run dev:studio
```

Unit/round-trip tests:

```bash
npm run test:studio
```

Полная интеграционная проверка из корня репозитория:

```bash
npm run verify:studio
```

Архитектура и список редакторов описаны в `docs/HONKNET_STUDIO.md`.
