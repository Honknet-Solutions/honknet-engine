# Honknet Game Template

Минимальный отдельный игровой репозиторий для Honknet Engine.

```bash
npm install
npm run typecheck
npm run build
```

Версии движка и SDK закреплены в `honknet.lock`. Игровой код находится в `server`, `client` и `shared`; YAML/FTL/RSI-контент — в `content`, `maps`, `resources` и `localization`.
